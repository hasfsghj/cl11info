# din ce consta fiecare folder?


## caiet
in acest folder sunt toate temele din clasa, scrise. include lucruri precum exemple de cod (pentru mine insumi), exercitii rezolvate, si, desigur, notite!

## important
acest folder este mai mult pentru mine. altora ar putea sa nu le fie trebuiasca, fiindca include lucruri ca transformari dintr-o baza in alta, sau formule.

## sem1 / sem2
pentru a fi mai usor de navigat, am creat 2 foldere aparte pentru fiecare semestru, anume cand vine vorba de rezolvarea problemelor. aici am lucrari de laborator, codul unor probleme de pe nerd arena, si alte lucruri similare.

## templates
algoritmi, precum scrierea si citirea dintr-un array. mai mult pentru convenienta mea, ca sa pot da copy paste la algoritmii des folositi.
